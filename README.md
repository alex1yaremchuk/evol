# evol

Статический сайт-презентация на русском языке про основные развилки в эволюции.

Внутри три схемы:

- от эукариотической клетки к амниотам;
- синапсидная ветка к млекопитающим;
- диапсидная ветка к динозаврам и птицам.

Открыть локально можно прямо через `index.html`. Для GitHub Pages достаточно опубликовать содержимое папки `evol` как статический сайт.

Содержимое презентации редактируется в `data.js`. `app.js` отвечает за отрисовку, навигацию и шкалу времени.

Фотографии лежат в `assets/photos`, авторы и лицензии перечислены в `CREDITS.md`.
